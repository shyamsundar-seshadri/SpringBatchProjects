# Spring-Boot Batch Rest White App

A White App for invoking Spring Batch Job from Spring Boot Rest Controller 
Jobs are configured annotation based
A simple CSV Reader is implemented

## Getting Started

### Prerequisites

Should know basics of Spring, Maven, etc 

### Start Point
DemoController Class for Rest Controller
BatchJob Class for Batch Job
